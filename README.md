# todo.fasm

To-Do Web Application in flat assembler for Linux x86_64

## Quick Start

Install [flat assembler](https://flatassembler.net/)

```console
$ fasm ./todo.asm
$ chmod +x ./todo
$ ./todo
open browser at http://localhost:6969/
```
